UoftHacks12 2025 Winners

Scribble

  • Tech Stack: React Native, Firebase, Mapbox, Midnight, Cohere, Groq
  • Github URL: Project Link

Scribble is a location-based note-sharing platform that lets users drop and discover "scribbles" at exact geographical points. Whether it's a quick thought, artwork, or local recommendation, users can leave their digital mark for others to see.

The platform integrates AI & ML to summarize clustered posts, filter inappropriate content, and recommend trending areas. A real-time map interface powered by Mapbox ensures a seamless exploration experience.

Features include personalized feeds based on geolocation, collaborative contributions to public notes, and dynamic notifications for trending "scribbles" nearby.
